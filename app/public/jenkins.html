Jenkins Tutorial

A minimal walk-through to teach Jenkins using this repository.

What you'll find here

  • A sample Jenkinsfile available at /api/jenkinsfile
  • An example pipeline trigger endpoint: /api/trigger-pipeline
  • Steps to configure a Jenkins job using this repo are below.

Suggested classroom exercise

  1. Fork or clone this repository into your SCM.
  2. Create a new Jenkins Pipeline job and point it at the repository. Choose "Pipeline script from SCM" and select your branch.
  3. Use the sample Jenkinsfile to run a basic Build/Test/Deploy sequence.
  4. Optionally, configure a webhook from your SCM to hit /api/trigger-pipeline to simulate external triggers.

Notes

This app also exposes a small UI to manually trigger a demo pipeline.
